
# Concerti

Questa cartella contiene l'elenco dei miei concerti suddivisi per anno in file CSV, oltre a un file totale che raccoglie tutti gli eventi.

Ho partecipato a più di 750 eventi musicali, dal Nord al Sud Italia e in molte città europee.

## Struttura della cartella

- `2016.csv`, `2017.csv`, ..., `2026.csv`: file CSV con i concerti di ogni singolo anno  
- `Concerti.csv`: file CSV che raccoglie tutti i concerti degli anni  

Questi file rappresentano un archivio dettagliato della mia esperienza live, utile per consultazioni, analisi e ricordi musicali.
